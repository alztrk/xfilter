# XFilter - Filter Verified Accounts on X.com

XFilter is a minimalist browser extension that allows you to filter and hide verified accounts on X (formerly Twitter). It provides control over your feed by hiding blue checkmark accounts while maintaining exemptions for accounts you already follow.

![XFilter Logo](icons/icon128.png)

## Features

- Hide Verified Accounts: Automatically filter out tweets and replies from verified accounts.
- Following Exemption: Keep seeing verified accounts that you actually follow.
- Whitelist: Always show specific accounts by adding them to your whitelist.
- Keyword Filtering: Hide tweets containing specific words or phrases.
- Multilingual Support: Support for English and Turkish.
- Local Cache: Optimized performance with local storage caching for seen accounts.

## Installation

Currently, XFilter is available for manual installation:

1. Clone this repository or download it as a ZIP file.
2. Open your browser and navigate to chrome://extensions.
3. Enable Developer mode.
4. Click Load unpacked and select the xfilter folder.
